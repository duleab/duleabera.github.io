Project Overview

This comprehensive financial sentiment analysis system processes financial news headlines to determine market sentiment and analyzes correlations with actual stock price movements. The project implements multiple machine learning approaches including traditional NLP techniques and modern transformer models.

Objective

Develop an intelligent system that can accurately predict stock market movements by analyzing the sentiment of financial news headlines using advanced NLP techniques.

Achievement

Successfully achieved 94.2% accuracy with DistilBERT model and established strong correlation (0.73) between news sentiment and stock price movements.

Methodology & Workflow

Data Collection

Gathered financial news headlines from multiple sources including Reuters, Bloomberg, and Yahoo Finance. Collected corresponding stock price data using yfinance API for correlation analysis.

  • 50,000+ financial news headlines
  • Stock data for 100+ companies
  • Time period: 2020-2024

Data Preprocessing

Comprehensive text cleaning and preprocessing pipeline to prepare data for machine learning models.

  • Text normalization and cleaning
  • Tokenization and lemmatization
  • Stop word removal
  • Feature engineering

Model Development

Implemented and compared three different approaches for sentiment analysis, each with unique strengths.

  • CNN: Fast processing with good accuracy
  • LSTM: Sequential pattern recognition
  • DistilBERT: State-of-the-art transformer model

Correlation Analysis

Statistical analysis to determine the relationship between news sentiment and stock price movements.

  • Pearson correlation coefficient calculation
  • Statistical significance testing
  • Lag analysis for timing effects

Visualization & Reporting

Created comprehensive dashboards and reports to present findings and model performance.

  • Model performance comparison charts
  • Correlation heatmaps
  • Time series analysis plots

  • Weekend news has stronger impact on Monday opening prices
  • After-hours news shows delayed correlation effects
  • Earnings-related news has immediate impact (same day)

Sector Analysis
